# Servomoteur

Le servomoteur permet un contrôle précis de l'angle mécanique (généralement de 0° à 180°).

![Servomoteur](/components/servomotor/servomotor.jpg)

## Câblage PWM

Le pilotage se fait via un signal PWM. Il est **critique** de relier les masses (GND) si vous utilisez une alimentation externe pour le moteur.

| Couleur Fil | Connexion | Note |
| :--- | :--- | :--- |
| **Rouge** | **Alim Externe (5V-6V)** | Ne pas tirer trop de courant sur l'ESP32 directement. |
| **Marron / Noir** | **GND** | Doit être commun avec le GND de l'ESP32. |
| **Orange / Jaune** | **GPIO (PWM)** | N'importe quel GPIO de sortie (ex: 14). |

> [!TIP]
> Si le servo a des comportements erratiques (jitter), vérifiez que votre alimentation externe est stable et que les masses sont bien reliées.

## Utilisation Code

L'implémentation `Servo` utilise le PWM hardware de l'ESP32.

```python
from src.Framework.Servo.Servo import Servo
from src.Framework.Servo.ServoDelegate import ServoDelegate

# Initialisation sur le GPIO 14
my_servo = Servo(pin_id=14, delegate=ServoDelegate())

# Positionner à 90 degrés
my_servo.set_angle(90)

# Couper le signal (relâche le moteur)
my_servo.off()
```
